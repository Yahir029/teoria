Herramientas en la Etapa de Analisis


Definicion

Estas herramientas ayudan a los especialistas en sistemas a documentar un sistema existente, ya sea este manual o automatizado, y a determinar los requerimientos de una nueva aplicación.


La elección del instrumento adecuado nos permitirá recopilar la información que sea capaz de dar respuesta a los objetivos de la investigación. Un estudio científico no tendrá éxito sin el apoyo de instrumentos de recolección de datos apropiados y válidos. La precisión de los datos se determinará a partir de la capacidad de un investigador para recopilar métodos de recopilación de datos.


Las Herramientas más populares

Zoho Analytics

Esta plataforma consiste en un software centrado en la inteligencia empresarial, la generación de informes y el análisis de datos.


Tableau Public

Esta plataforma permite procesar información, darle forma y más adelante presentarla a través diferentes herramientas de visualización de datos.
La herramienta no requiere de conocimientos de código ni de lenguajes específicos de programación.


Rapid Miner

Rapid Miner consiste en una plataforma de análisis de datos y Machine Learning
Este software permite integrar información de diferentes fuentes y tipologías, incluyendo datos desde Access, Excel, Microsoft SQL, entre otros. Además, incorpora funcionalidades Machine Learning para generar una plataforma lo más intuitiva posible


Knime

Este software genera un entorno de código abierto para el análisis de datos, convirtiéndose en una de las herramientas de análisis de datos más populares y completas.
Así, entre sus funcionalidades incluye instrumentos para la detección de anomalías en los procesos de la empresa, la optimización de los inventarios o la medida de la satisfacción del cliente.


Qlick Sense

Ofrece además aplicaciones de IA que, a través de tecnología asociativa, muestran sugerencias a los usuarios para potenciar el descubrimiento de tendencias y oportunidades ocultas en la información. Entre las ventajas que ofrece destaca su uso intuitivo, de modo que no son necesarios conocimientos de programación para sacar partido a los datos.


Herramientas para la colección de datos

Datos Primarios

Los datos primarios son datos obtenidos directamente de fuentes de información directamente.

Datos Secundarios

Los datos secundarios son datos obtenidos no directamente de fuentes de información. Hay una gran cantidad de datos secundarios repartidos por todo el mundo.


Herramientas para diagramación

Un diagrama de estructura visualiza cómo funciona un sistema desde la entrada inicial, hasta el procesamiento y, finalmente, hasta la salida deseada.
Es especialmente útil para determinar todas las interfaces involucradas entre las diferentes partes y ayuda a los desarrolladores a acordar cómo se debe conectar cada parte según los modelos que se muestran en el diagrama de estructura.

Diagrama de estructura de datos

un modelo de datos conceptual que se utiliza para documentar las entidades en una base de datos, sus relaciones y las restricciones asociadas con ellas.

Diagrama de estructura completa

un tipo de diagrama de estructura estática que muestra la estructura interna de una clase y sus colaboraciones con otras clases y objetos.


Herramientas para Diccionario

Un diccionario de datos es una colección de descripciones de los objetos o elementos de datos en un modelo de datos para el beneficio de los programadores y otras personas que necesitan consultarlos.


Diccionario de datos activos

Estos son diccionarios de datos creados dentro de las bases de datos que describen y reflejan automáticamente cualquier actualización o cambio en sus bases de datos anfitrionas.

Diccionario de datos pasivos

Estos son diccionarios de datos creados como bases de datos nuevas, separadas de las bases de datos que describen, con el fin de almacenar información del diccionario de datos.

Gráficos de cuadrícula

Los gráficos de cuadrícula se utilizan para representar la relación entre dos conjuntos de factores en un método tabular. Un análisis de gráfico de cuadrícula es útil para erradicar informes innecesarios o elementos de datos innecesarios de los informes.


Diagrama de flujo del sistema

Un diagrama de flujo del sistema es una representación pictórica o esquemática del flujo lógico de operaciones e información de una organización. Representa la clara relación entre el procesamiento de entrada y la salida considerando todo el sistema.


Árbol de decisión

Algunas decisiones implican una serie de pasos. El resultado de la primera decisión guía la segunda; la tercera decisión depende del resultado de la segunda, y así sucesivamente.


Simulación

El modelo de simulación describe la operación del sistema en términos de eventos individuales, componentes del sistema.
Principalmente, implica el desarrollo de un modelo que es principalmente de naturaleza matemática en lugar de describir directamente el comportamiento del sistema en general.


Mesas de decisión

Las tablas de decisiones son un método gráfico para representar una secuencia de decisiones lógicas. Se prepara en forma tabular. Enumera todas las condiciones posibles y el conjunto de acciones asociado. Una tabla de decisiones consta de cuatro partes: stub de condición, entradas de condición, stub de acción y entradas de acción.
